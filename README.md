C2 Framework Project (Proof of Concept)
Proyecto educativo de un Framework de Comando y Control (C2) para entender la comunicación entre implantes y servidores en entornos Windows.

🚀 Funcionalidades
Agente C++ Multihilo: Ejecución de comandos remotos (RCE) con persintencia.

Servidor Multicliente: Gestión de múltiples víctimas simultáneas usando Python threading.

Post-Explotación: Keylogger de bajo nivel integrado con exfiltración de datos mediante comando dump.

Evasión Visual: Compilación en modo ventana oculta (-mwindows).
